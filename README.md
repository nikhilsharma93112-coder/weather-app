A minimalist, real-time weather application that provides instant forecast data through a clean and intuitive user interface.

Features
Real-time Updates: Fetch current weather conditions for any city.

Simple UI: Clean, distraction-free interface for quick data reading.

Responsive Design: Works seamlessly on mobile and desktop devices.

Detailed Metrics: Displays humidity, wind speed, and temperature.

Tech Stack
JavaScript

React

OpenWeatherMap API

CSS

Setup
Clone the repo

npm install

Create a .env file and add your API key

npm start
